def plot_gaussian_mixture(clusterer, X, show_centroid=True, show_contour=True, show_boundary=False):
    mins = X.min(axis=0) - 0.1
    maxs = X.max(axis=0) + 0.1
    x, y = np.meshgrid(np.linspace(mins[0], maxs[0], num=1000),np.linspace(mins[1], maxs[1], num=1000))
#     a = []
#     for i in range(len(x)):
#         for j in range(len(x)):
#             a.append([x[i][j],y[i][j]])
       
    # Centroids 시각화
    if show_centroid:
        centroids = clusterer.means_# <your code> to use the mean of each components to get the centroids
        plt.scatter(centroids[:, 0], centroids[:, 1], marker='o', s=20, linewidths=8, color='w', zorder=10)
        plt.scatter(centroids[:, 0], centroids[:, 1], marker='x', s=20, linewidths=1, color='k', zorder=11) 
    
    # 밀도 등고선 시각화
    if show_contour:
        Z = -clusterer.score_samples((np.c_[x.ravel(), y.ravel()]))# <your code> to compute the negative log-likelihood of each sample
        Z = Z.reshape(x.shape)
        cntr = plt.contourf(x, y, Z, norm=LogNorm(vmin=1.0, vmax=40), levels=np.logspace(0, 2, 10))
        plt.contour(x, y, Z, norm=LogNorm(vmin=1.0, vmax=40), levels=np.logspace(0, 2, 10), linewidths=1, colors='k')
    
    # 결정 경계 시각화
    if show_boundary:
        Z = clusterer.predict((np.c_[x.ravel(), y.ravel()]))# <your code> to predict the labels
        Z = Z.reshape(x.shape)
        plt.contour(x, y, Z, linewidths=2, colors='r', linestyles='dashed')
    
    plt.xlabel("$f_1$")
    plt.ylabel("$f_2$", rotation=0)

Discussion¶

1) 모수적 밀도추정과 비모수적 밀도추정을 비교하여 설명하세요.

[답변작성]

모수적 밀도추정의 방법은 확률분포가 어떠한 파라미터를 갖는 확률분포를 따른다고 가정하고 그 확률분포의 파라미터를

ML Estimation이나 MAP Estimation을 통하여 추정하는 방법이다. 또한 파라미터 개수가 유한하다.

그리고 비모수적 밀도추정은 확률분포가 임의의 형태를 띤다고 생각하고 즉, 유한개의 파라미터를 정할 수 없다고 생각하고

샘플의 분포를 추정하는 방식이다. 대표적인 방식으로는 파젠창, k-최근접 이웃 추정 방법 등이 있다.

2) EM 알고리즘을 이용한 GMM 밀도추정 방법에 대해 설명하세요.

[답변작성]

처음 GMM의 경우에는 파라미터를 모르므로 난수로 초기화 하거나 k-means clustering의 결과를 이용하여 초기화 한다.

그 후 E단계에서 각 샘플의 소속 정보를 추정하고 샘플이 어디에 속할지 확률을 파악한다.

다음 M단계에서는 각 분포에 소속된 샘플들을 이용하여 파라미터(가우시안)를 갱신한다. 이때 각 샘플이 독립이라면 각 확률에

log를 씌워 더해주는 곱해주는 방식으로 ML Estimation을 진행가능하다.

Stop condition을 만족할때까지 EM을 반복해준다.

3) K-means를 이용한 군집화와 GMM을 이용한 밀도추정 기반의 군집화의 차이에 대해 설명하세요.

[답변작성]

K-means를 이용한 군집화에서는 각 샘플이 어느 군집에 속하는지 결정하고 군집의 중심을 update했다. 이때 각 샘플은 하나의 군집에만 들어가야 한다. GMM을 이용한 밀도추정의 경우 샘플이 어느 분포에 속할지를 추정하는 방법이다. 이때 각 샘플은 어느 분포에 속할지 확률로 나타내 진다.

4) GMM의 Covariance type를 다른 type ('tied', 'spherical', 'diag')로 바꿀 수 있습니다. 또한, default Covariance type는 'full'로 클러스터의 모양, 크기, 방향에 제약이 없습니다. 그러면 Covariance type를 바꾸면 클러스터가 어떻게 바뀌는지 수행해보고, 이를 설명하세요.

[답변작성]

가우시안의 경우 평균과 공분산 행렬로 결정이된다. 직접 네가지 경우를 해봤을 때 spherical과 diag는 거의 비슷한 모양이 나왔다. 둘 다 거의 각 분포가 원형에 가깝게 나왔고 가까운 두개의 분포의 밀도선은 합쳐졌다. tied의 경우에는 조금은 달랐는데 full로 했을때와 diag의 사이로 보였다. 각 분포가 원형은 아니었고 살짝씩 각자의 분포를 띠고 있었다.

찾아보니 full의 경우에는 각자 다른 모양으로 각자 다른 위치를 가진다고 했고 tied는 같은 모양을 가지지만 모양을 알 수 없다고 했다. spherical의 경우에는 circular한 형태를 띤다고 했다.
